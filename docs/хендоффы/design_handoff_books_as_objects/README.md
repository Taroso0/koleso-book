# Handoff: Книги как объекты (секция «Книги» на Витрине)

## Overview
Переработка секции `#books` на хабе-лендинге «Витрины». Дефолтные карточки-списки с «Читать →» заменяются **editorial-разворотами**: каждая изданная книга подаётся как физический объект (обложка + колонка сведений), третья книга («в работе») — намеренно НЕ объект, а стопка черновиков, ведущая в Мастерскую. Реализует §9 концепции: уважительная подача книг как главных экспонатов Витрины.

## About the Design Files
Файл `Шаг 7 - Книги как объекты.html` в этом бандле — **дизайн-референс, созданный в HTML**: прототип, показывающий задуманный вид и поведение, а НЕ продакшн-код для копирования. Задача — **воссоздать этот дизайн в существующем окружении кодовой базы** (Next.js App Router + React + Tailwind + shadcn/ui, как в проекте «Колесо»), опираясь на её паттерны: компоненты `Reveal`, `AccentLine`, `FogReveal`, класс `paper-surface`, токены `font-serif`/`font-mono`/`font-sans`, палитра из `globals.css`, данные из `content/books/*.json` и `content/themes.ts`.

Обложки в прототипе нарисованы CSS-типографикой (без картинок) — это осознанный **плейсхолдер**, а не финал. При наличии реальных ассетов обложек они ставятся вместо макетов (см. Assets).

## Fidelity
**High-fidelity.** Финальные цвета, типографика, отступы и композиция. UI воссоздать точно, но через существующую систему проекта (Tailwind-токены и компоненты), а не хардкодом hex. Значения ниже даны как ориентир — где в проекте уже есть токен (`--sodium`, `paper-surface`, `font-serif`), используйте его.

## Целевой файл
Секция живёт в `Колесо/app/(vitrina)/page.tsx`, блок `<section id="books" …>` (в текущем виде — заголовок + `books.map()` в `grid sm:grid-cols-2` с `<Link>`-карточками). Его нужно заменить. Данные книг уже приходят из `getBooks()` (`content/books/*.json`); темы — из `content/themes.ts`. Маршрут чтения существует: `/read/${book.id}`.

## Screens / Views

### Секция «Книги» (один вертикальный поток внутри `max-w-5xl` контейнера)
**Назначение:** показать две изданные книги как объекты и анонсировать третью в работе.
**Layout:** заголовок «Книги» + акцентная линия + лид-абзац; ниже — три блока подряд с большими вертикальными отступами (~76–96px между блоками):
1. Разворот 1 — обычная композиция (обложка слева, сведения справа).
2. Разворот 2 — **зеркальная** (сведения слева, обложка справа).
3. Полоса «третья книга» — отделена верхней hairline-границей; слева стопка черновиков, справа сведения.
Внизу — «кода»: две моно-строки (`littera scripta manet …` слева, `2 изданы · 1 в работе` справа) над/под hairline.

**Grid разворота:** desktop `grid-template-columns: 340px 1fr` (зеркальный — `1fr 340px` + смена порядка колонок), `gap: clamp(32px, 6vw, 72px)`, `align-items: center`. На ≤760px — одна колонка, обложка сверху, `gap: 36px`.

#### Компонент: Обложка-объект (`.book`)
- Размер: `width: min(300px, 72vw)`, `aspect-ratio: 5 / 7.2`, `border-radius: 2px 4px 4px 2px`.
- Физичность БЕЗ 3D, чистый CSS:
  - Тень книги: `box-shadow: 0 1px 0 oklch(1 0 0 / .5) inset, 1px 0 0 oklch(0 0 0 / .06) inset, 0 24px 34px -22px oklch(.2 .02 250 / .55), 0 6px 10px -6px oklch(.2 .02 250 / .3)`.
  - **Корешок** (`::before`, ширина `--spine: 14px`, слева): вертикальный градиент сгиба `linear-gradient(90deg, oklch(0 0 0 / .16), oklch(0 0 0 / .02) 45%, oklch(1 0 0 / .22) 72%, oklch(0 0 0 / .1) 100%)`.
  - **Блок страниц** (`::after`, справа, `-5px` наружу, ширина 5px): `repeating-linear-gradient(180deg, oklch(.99 .004 95), … 2px, oklch(.82 .008 95) 3px)` + лёгкая тень. Имитирует торец страниц.
  - Контент обложки (`.cover`) с внутренним отступом слева `calc(var(--spine) + 14px)`, чтобы не залезать на корешок.
- Типографика обложки (центрирована): издатель/автор — `font-mono` 9px, letter-spacing .22em, uppercase, opacity .65; титул — `font-serif` `clamp(1.7rem, 2.6vw, 2.15rem)`, weight 500; тонкая линейка 34×1px; подзаголовок «рассказы» — курсив serif .85rem; год римскими — `font-mono` 10px.
- Две скины:
  - **Манила** (`Колизей`): `linear-gradient(160deg, var(--manila) oklch(.915 .028 92), var(--manila-deep) oklch(.86 .035 90))`, текст `--manila-ink` oklch(.32 .025 70).
  - **Бетон** (`Колесо Сансары`): `linear-gradient(165deg, oklch(.37 .012 250), oklch(.33 .011 250) 55%, oklch(.29 .011 252))`, текст `--concrete-ink` oklch(.9 .008 95); монограмма-глиф (кружок 30px с точкой) в цвете натрия `--sodium`.
- Под обложкой — моно-пометка `макет обложки · заменить реальным ассетом` (uppercase, 9px, `--faint`). Удалить, когда появится реальная картинка.

#### Компонент: Колонка сведений (`.info-col`)
- `книга 01 / 03` — `font-mono` 10px, letter-spacing .2em, `--faint`.
- Заголовок книги (`<h3>`) — `font-serif` 1.9rem, weight 500, tracking -.01em.
- Метаданные (моно, «система»): `font-mono` 11px, uppercase, `--ink-soft`; жирные значения `--ink`. Пример: **2022** · 17 рассказов · издана. Число рассказов = `book.stories.length`.
- Фраза издателя (`.pubsay`) — человеческий текст, `font-serif` 1.06rem, line-height 1.65, `--ink-soft`; выделения — `<em>` в цвете `--ink`.
- Темы Колеса (`.themes`): подпись `стоит на темах Колеса` (моно 9.5px) + список `<a>`-«пилюль» (`font-sans` .82rem, padding 5×12, border `--hairline`, radius 999px). **Ссылки ведут в Колесо** (`#wheel` в прототипе; в проекте — якорь графа `/#wheel` или на конкретную тему). hover: border → `--sodium`, текст → `--ink`.
- Действие чтения (`.read`) — ОДНО ясное: `font-sans` .95rem, `border-bottom: 1px solid var(--sodium)`, padding-bottom 3px; hover — цвет `--sodium`. Ведёт на `/read/${book.id}`.

#### Компонент: Стопка черновиков (третья книга, `.stack`)
- `width: min(300px, 72vw)`, `aspect-ratio: 5 / 6`. Три листа `.sheet` внахлёст, каждый со своим лёгким поворотом (переменная `--draft-tilt: 2deg`) и сдвигом:
  - `s1` — манильский, повёрнут `-tilt`, сдвиг (-6px, 8px).
  - `s2` — белый, повёрнут `+tilt*.6`, сдвиг (8px, 3px).
  - `s3` (верхний) — белый с контентом: моно-тег `черновик · фрагмент 07` в цвете натрия; курсивный фрагмент с зачёркнутой строкой (`text-decoration: line-through`, цвет линии полупрозрачный ink); ниже — «линованный» блок (`repeating-linear-gradient` в hairline).
  - Скрепка `.clip` сверху — рамка 14×34px, radius 7/3, без нижней грани, opacity .65.
- Колонка сведений: `книга 03 / 03 · в работе`, `<h3>Без названия</h3>`, мета `пишется сейчас · фрагменты и заметки`, фраза, действие `Зайти в Мастерскую →` (`/workshop`). НЕ обложка — визуально это черновик, а не объект.

## Interactions & Behavior
- **Появление в кадре:** обернуть блоки в существующий `Reveal` (как сейчас в page.tsx). Движется только проявление (§10), всё под `prefers-reduced-motion` / `ReduceMotionToggle`.
- **Hover обложки:** лёгкий подъём `hover:-translate-y-0.5` (как у текущих карточек) — опционально; главное действие — ссылка «Читать».
- **Hover тем-пилюль:** border и текст переходят к натрию/ink за .2s.
- **Навигация:** темы → Колесо (`#wheel`), «Читать» → `/read/${id}`, «Зайти в Мастерскую» → `/workshop`.
- **Responsive:** ≤760px все развороты схлопываются в одну колонку (обложка сверху, сведения снизу), зеркальность отключается.

## State Management
Состояния нет — статическая секция на сборке (SSG). Данные приходят из `getBooks()` и `themes` в момент рендера `page.tsx`. Твики из прототипа (зеркальность, скин, толщина корешка, наклон) — это **дизайн-решения**, не рантайм-фичи: зафиксируйте выбранные значения (mirror=on для 2-й книги, скин «бетон» для «Сансары», `--spine: 14px`, `--draft-tilt: 2deg`) как константы стиля.

## Design Tokens
Цвета (oklch — совпадают с духом `globals.css`; используйте существующие токены проекта где есть):
- `--paper-bg` `oklch(.968 .006 160)` — фон
- `--ink` `oklch(.235 .012 250)` — основной текст
- `--ink-soft` `oklch(.47 .012 250)` / `--faint` `oklch(.6 .01 250)`
- `--sodium` `oklch(.7 .14 78)` — **единственный тёплый акцент** (линия, подчёркивание действия, hover, глиф). Больше нигде.
- Манила: `--manila` `oklch(.915 .028 92)`, `--manila-deep` `oklch(.86 .035 90)`, `--manila-ink` `oklch(.32 .025 70)`
- Бетон: `--concrete` `oklch(.33 .011 250)`, `--concrete-ink` `oklch(.9 .008 95)`
- `--hairline` `oklch(.235 .012 250 / .14)`
Типографика: serif = `font-serif` (Source Serif 4) — человеческий текст, заголовки; mono = `font-mono` (JetBrains Mono) — все метаданные/счётчики/годы; sans = `font-sans` (Inter) — действия и пилюли.
Радиусы: книга `2px 4px 4px 2px`; пилюли `999px`; листы `1px`.
Прочее: `--spine: 14px` (корешок), `--draft-tilt: 2deg` (наклон черновиков).
