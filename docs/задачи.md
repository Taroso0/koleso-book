## Требуют решения владельца

- [ ] **Узел-тема «Колеса» объявлен кнопкой, но нажатие ничего не делает.**
      `WheelCanvas.tsx:393` ставит `role="button"` узлам-темам, а `activate()`
      (`:262-263`) выходит сразу для всего, кроме рассказов. Скринридер объявляет
      кнопку, читатель жмёт Enter — тишина. Фактически «действие» у темы уже
      происходит на ФОКУСЕ (подсветка созвездия) — это паттерн «выбор следует за
      фокусом», а не кнопка. Оба варианта меняют поведение графа, а правки «Колеса»
      идут через план и подтверждение:
      1) дать теме действие — Enter ведёт к её секции в доступном индексе
         (`#theme-<id>`, нативный `<details>` там раскрывается по якорю);
      2) сменить роль на невзаимодействующую и описать состояние через `aria`.
- [ ] **`.mdx` — не MDX.** Компилятора нет; тело рендерит `ProseBody` (абзацы,
      `*курсив*`, `***`). Любой заголовок или JSX отрендерится текстом. Решить:
      переименовать в `.md` и задокументировать поддерживаемую разметку — либо
      подключить MDX осознанно.
- [ ] **Чип «Мастерская» в `BookSwitcher`** (опционально). Раздел `/workshop` — часть
      «Витрины», а не книга чтения, поэтому в переключатель не добавлен. Зафиксированная
      возможность: неактивный чип-ссылка «Мастерская →».

## Инженерный долг

Найдено архитектурным аудитом 2026-07-21, отложено сознательно.

**Функциональные дыры**

- [ ] **Рассказ короче экрана не отмечается прочитанным.** В «свитке» прогресс —
      `scrollY / (scrollHeight − innerHeight)`; если текст помещается на экран,
      знаменатель ≤ 0 → `pct` всегда 0, и «Колесо» рассказ не запомнит. Тот же случай
      в постраничном режиме при `pageCount === 1`. Ни один из 34 рассказов сейчас так
      не выглядит, но дыра реальна. → `ReaderShell.tsx`, эффект скролла
- [ ] **Reduced-effects: 1 кадр движения при F5.** Ручной тумблер читается из
      localStorage в эффекте. Паритет при этом цел (итог верный). Низкий приоритет.
      → `MotionProvider.tsx`

**Условия приёмки третьей книги** (делать вместе с ней, не раньше)

- [ ] **Коллизия slug.** Каталог `content/stories/` плоский, slug не содержит книгу:
      рассказ №17 «Дорога» в новой книге молча перезапишет `17-doroga.mdx` из «Колеса
      Сансары».
- [ ] **Порядковый номер внутри slug** (`ingest_pdf.py:96`). Slug служит именем файла,
      ключом манифеста, URL И ключом localStorage прогресса читателя. Вставка рассказа
      в середину книги сдвигает `order` у последующих: осиротевшие `.mdx` продолжат
      грузиться, `themes[]` обнулятся, прогресс читателей потеряется. Миграция на
      семантический slug правильна, но требует редиректов.

**Вес и скорость**

- [ ] **three + R3F + drei ради одного шейдера.** Чанк `three` — 864 КБ на 104 строки
      `GrainCanvas`. Сырой WebGL даст тот же визуал. Самый крупный резерв «Витрины».
- [ ] **`gsap` + `lenis` в initial-бандле `/contacts` и `/workshop`**, где скролл-сцен нет.
- [ ] **d3-force в главном потоке на hover.** `WheelCanvas.tsx:165-176` синхронно в теле
      рендера гоняет 80 итераций (200 при смене множества прочитанного).
- [ ] **Плашки в 3–5 раз крупнее, чем рисуются.** 24 МБ в git: 34 файла по ~710 КБ,
      1156–1392 px, при отрисовке 384 px. Ресайз до ~800 px срежет ~20 МБ.
      Отдельно: WebP байтово недетерминирован (зависит от версии Pillow/libwebp) —
      смена версии даст diff по всем 34 бинарникам.

**Инфраструктура**

- [ ] **Python-пайплайн без гейта.** 615 строк, перезаписывающих ручной труд, не покрыты
      ничем. Минимум — `ruff` + `compileall` в CI; дальше 3–4 pytest на `build_mdx.py`
      (сохранение `themes[]` и галочек) на фикстурах, без PDF.
- [ ] **`ds-bundle` отстал на коммит.** Маркер `_ds_needs_recompile` стоит;
      `.wheel-label-warm` и зерно `.wheel-night` во внешнем DS-проекте доиюльские.
      Пересборка выгружает бандл на сторонний сервис — нужно явное согласие владельца.
- [ ] **`lib/og.tsx:13` читает `.woff` из `node_modules/@fontsource` по внутреннему
      пути** — сломается на pnpm, при prune и при минорном обновлении пакета. Плюс
      палитра OG — захардкоженный sRGB-дубль OKLCH-токенов (`:31-36`).
- [ ] **`app/globals.css` — 1066 строк монолита** с покомпонентными стилями; контракт
      JS↔CSS держится на ~20 магических строках (`data-live`, `data-zachin-boot`,
      `.wheel-link`, `.reading-root`) без типов и без единого места объявления.
- [ ] **Tailwind сканирует `.design-sync/previews` и `shims`** — закрыто целиком через
      `@source not`, но каталог остаётся в рабочем дереве; при добавлении новых файлов
